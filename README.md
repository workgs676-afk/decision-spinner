# 🎡 Decision Spinner

A simple and fun web app that helps you make decisions by spinning a wheel.

## 🚀 Features

* Add multiple options
* Spin the wheel to randomly select one
* Smooth spinning animation
* Clean and minimal UI

## 🌐 Live Demo

https://workgs676-afk.github.io/decision-spinner/

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📦 How to Use

1. Enter your options (one per line or input)
2. Click the **Spin** button
3. Let the wheel decide 🎯

## 💡 Idea

Built to make decision-making fun and remove overthinking.

## 🔮 Future Improvements

* Sound effects while spinning
* Remove/edit options
* Save options using localStorage
* Better animations and themes
